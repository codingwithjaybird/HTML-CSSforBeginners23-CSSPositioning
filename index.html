CSS Positioning

A POSITION FIXED (Top, Right, Bottom or Left) element doesn't move and is positioned relative to the viewport! It stays fixed to that corner even when the user scrolls the webpage.

POSITION STICKY! A sticky element stays "stuck" to the top of the page once the user scrolls down.
